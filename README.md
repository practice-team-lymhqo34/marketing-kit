# 📦 LogiFlow — Marketing Kit

This repository contains all visual, marketing, and presentation materials for the **LogiFlow** project. It serves as a centralized hub for assets needed to prepare the Pitch Deck and promotional campaigns.

## 📂 Repository Structure

* **`/branding/`** — Brand Identity: primary and white logos (SVG/PNG), Hero Banner, and technical Style Guide (PDF).
* **`/assets/`** — Visual Evidence: static UI screenshots (desktop and mobile views) and GIF animations of key flows (Teaser Assets).
* **`/video/`** — 60-second product promo video (The Solution in Action) and video thumbnail.
* **`/copywriting/`** — Text materials and presentation scripts (Elevator Pitch).
* **`/strategy/`** — Business Analytics: SWOT analysis, user personas, monetization model (Pricing Strategy), and Go-to-Market strategy.
